# Расписание лекций

Структура:
- `src/` — исходные фалы проекта
- `build/` — собранный проект

## Требования
- node v.6.10

## Установка проекта

```bash
$ git clone https://github.com/Lunory/Timetable.git project
$ cd project
$ npm install
```

## Запуск проекта

```bash
$ npm start
```
